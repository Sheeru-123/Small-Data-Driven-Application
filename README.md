##  Overview

This dashboard lets users:
- Navigate between **Overview, Orders, and Settings** sections without page reload.
- View a table of orders.
- Search orders by customer name in real time.
- Filter orders by **status** (All / Pending / Shipped / Cancelled).
- Sort orders by clicking on table headers.

Built with:
- Semantic HTML
- Bootstrap 5 for responsive layout
- Custom CSS for theme overrides and hover effects
- Plain JavaScript for SPA behavior and data logic
